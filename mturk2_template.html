Instructions
You must be using Google Chrome to do this study. Please do NOT do this study if you have recently cleared your browsing history, or if this is not the computer and web browser you use for your everyday browsing activity.
In this study, we are looking to see if we can predict your personality based on your browsing patterns. You will first install a Chrome extension that will look at which sites you spend the most time on, and ask you some questions about how you use those sites. Then you will answer some general personality-related questions. The questions will take 10-20 minutes to answer total.
(The mechanical turk terms of service permit HITs that require software so long as it is not harmful. The extension is open-source and the results will be shared with only the Stanford HCI Research Group. You can uninstall the extension once you are done with this HIT.)
If you have any questions or issues, you can contact geza@cs.stanford.edu

Make sure to leave this window open as you complete the survey. When you are finished, you will return to this page to paste the code into the box.
